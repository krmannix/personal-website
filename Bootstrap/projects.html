Jetpack.
Jetpack is a port of the popular mobile phone game 'Jetpack Joyride' set in space. The game was implemented using Verilog, a Hardware Description Language, and is played with a Field Programable Gate Array. The user must avoid randomly generated blocks (space junk) of various sizes that fly accross screen from right to left. The spaceman can only move vertically, and is in constant horizontal movement to the right.

Pokejack.
A mash-up of Pokemon and Blackjack for Android. A player can play Blackjack traditionally, with doubling-down and betting. The dealer will decide when to hit, and shuffles the deck after every hand. The cards are Pokemon-themed, with each suit having a specific type of Pokemon. A user can win or lose the game based on how much money they currently have.

Multicycle CPU.
Implemented using Verilog, a Hardware Description Language, this project emulates a 5 stage Central Processing Unit. Each stage is broken down into smaller modules to more easily implement the logic behind the design. A controller is also implemented to run the control lines between stages. A graphic representation of the design is also included in the Github repository.

SmartMart.
In this paper, the research team presents a feasibility study that leverages the Internet of Things (IoT) technology to make store items “smart” so that they can automatically register and update their location information in an information retrieval system, allowing shoppers to search, locate, and map them on the store floor plan using mobile devices. A free-accessible Android-based mobile app SmartMart has been developed to demonstrate the promise of this preliminary work. Continuous development of this research could lead to a complete change in our day-to-day shopping experience.

SwingWord.
Developed as a Java Swing applet, SwingWord is a tool for elementary school teachers to aid in instructing young students phonetic sounds. The applet includes a library of over 70,000 words each broken down by phonetic sound in a variety of codes - teachers can select a single or pattern of phonetic sounds in a query, which will return all words in the library that exhibit these phonetic segments. The goal of this project is to provide a variety of words teachers can use as examples of phonetic sounds, giving students a broader range of possible words to learn English from.

Red, Write & Blue.
Red, Write & Blue is a mobile application currently designed for Android that enables everyday users to send motivational messages to troops overseas. Users will write messages that go through a series of anonymous peer reviews to ensure that the content is both appropriate and written with good intent. Users will be rewarded with coupons and deals through sponsers of the app. The application is finished, but the process of enabling the messages to be sent to troops & gaining sponsers is still in development.

C-Shell.
This is a short project to emulate a bash shell in C. The shell can interpret basic commands and perform the actions to the file directories using piping and redirection.
